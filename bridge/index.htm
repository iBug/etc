    概述

    第一部分介紹被OKBridge眾多橋牌愛好者使用的標準美國黃卡(SAYC)叫牌體系. SAYC是開叫五張高花的叫牌體系,但牌手可以根據自己的判斷在第三座開叫比較好的四張高花. 有關SAYC的許多自然的後續叫品, 牌手們可以賦予其為"迫叫", "邀叫", 或者"不迫叫"等含義.

    第一部分中所有的虛叫/約定叫都屬于標準的美國黃卡. 初學者可以和他/她的同伴約定不使用其中的部分約定叫, 但同伴間必須明確地同意哪些約定叫不包括在他們的叫牌中. 建議初學者可省略掉以上帶有 *號的約定叫. 在你逐漸掌握了SAYC的基本叫牌法後可一個一個的加入這些約定叫牌法.

    基本叫法
    1. 正常開叫五張或五張以上的高花(黑桃和紅心).

    2. 等長套( 5-5 或 6-6), 正常開叫高級別花色. 花色的級別從高到低分別是黑桃, 紅心, 方塊, 梅花.

    3. 4-4低花套, 正常開叫 1♦ .

    4. 3-3低花套, 正常開叫 1♣ .

    5. NT 開叫須是平均牌型, 但可以有五張高花或低花套.

    6. 1NT = 15-17HCP

    7. 2NT = 20-21HCP

    8. 3NT = 25-27HCP {見第二部分 賭博性 3NT}

      　

    9. 強 2♣ 開叫 = (22+ HCP)

      　

    10. 方塊, 紅心, 黑桃的弱二開叫. (5-11總牌點, 6 張套)

    對1NT開叫的應叫及後續叫品

    斯台曼(Stayman)

    對同伴1NT開叫後的2♣ 應叫是斯台曼. 這一應叫表示有8+ HCP并且至少有一個四張高花. 這一叫品被用來詢問1NT開叫方是否有四張套的高花. 1NT開叫方必須再叫 2♦ (否認有四張套的高花), 2♥, or 2♠. 如果答叫方再叫任一三階低花套, 表示有滿貫興趣并且所叫的低花套至少5張.

    如果答叫方有5/4或4/5的高花套, 在1NT開叫方否認有四張高花或所叫的高花是答叫方只有四張的那門, 答叫方可以再叫較長的高花套來表示自己的這門高花是5張: 也就是說, 1N-2♣-2♠-3♥(答叫方有5張紅心, 4張黑桃) 或 1NT-2♣-2♦-2♥(答叫方有5張紅心, 4張黑桃)

    注 1: 如果1NT開叫方有4-4雙高花, 1NT開叫者要先應 2♥, 如果答叫方的套是♠的話, 他將會再叫 2NT, 開叫方如覺得有將定約更好的話, 他可以再叫出他的4張 ♠. 例如:

    開叫方 答叫方 表示牌情
    1NT 2♣ 開叫方有4/4高花, 低限/高限 NT 開叫
    (不迫叫)
    2♥ 2NT
    3♠/4♠ Pass

    注 2: 當答叫方少于8HCP, 并準備PASS開叫方的任何再叫, 例如答叫方的牌型是 4-4-4-1 (梅花單張), 答叫方也可以采用斯台曼(Stayman)并常常能找到最好的部分分定約. 但同伴間必須互相知道2♣應叫可能是這種情況以免錯誤判斷加倍懲罰對方的競叫. {又見 低花斯台曼}

    第四花色迫叫 (4sf) ( ! ) *

    當答叫方沒有pass過, 在對方未加入叫牌的情況下再叫第四花色, 此叫品是進局迫叫并可以是虛叫.(例如 1♦-P-1♥-P-1♠-P-2♣!-?). 答叫方用第四花色來表明自己的牌是好牌, 12+ 總牌點, 但無好的再叫. 開叫方可用自然叫品進一步描述自己的牌情. 例如上述的叫牌過程, 開叫者可以作如下再叫:

    3. 在競叫中

    強2♣開叫

    2♣開叫一般表示有22+ 總牌點, 迫叫到3階高花或4階低花. 如開叫方在2♦應叫後再叫2NT, 表示有23-24HCP, 平均牌型, 後續叫品可采用2NT開叫的後續叫品. 對2♣ 開叫的HCP要求也可有例外. 以下準則可供參考: 如果是平均牌型, 要求至少22HCP;  如果是有9張贏墩的單套牌, 要求至少17HCP; 如果只有8張贏墩, 要求至少21HCP.

    定量4NT

    在1NT/2NT開叫後, 答叫方的直接加叫到4NT是定量4NT, 邀請開叫方叫6NT如是高限, pass如是低限. 定量4NT還適用于同伴在對方開叫後直接競叫1NT并且右手方的對手沒有叫牌.

    DOPI *

    "X表示沒有A, Pass 表示有1個A". DOPI可用于對手在黑木問叫後干擾的情況下應叫.

    具體叫法:
    X 0 A
    Pass 1 A

    控制張扣叫

    在王牌已經同意的情況下, 控制張扣叫可用于表示第一輪或第二輪的控制. 控制張扣叫一般是在3階(如果同意的王牌是高花)或4階(如果同意的王牌是低花)不跳叫的叫出從沒叫過的花色. (不要將試探性叫品和控制張扣叫搞混, 例如, 1♥-P-2♥-P-3♣ 是進局試探性叫品, 常常是短套, 詢問答叫方在此花色是否有幫)

    控制張扣叫的一般規則:

    1. 永遠不要pass同伴的控制張扣叫.

    2. 如有兩控制張, 一般先叫占用叫牌空間最小的花色. 在某些情況下則需要反轉扣叫的次序, 例如, 4♣-4♥-4♠ 而不是 3♠-4♥-5♣.

      　

    3. 首先扣叫同伴雙方都沒叫過的花色上的控制. 扣叫的優先次序是先A, 第二是缺門,第三是K. 一個例外就是先叫從沒叫過的花色上的第二輪控制, 再叫某一門已經叫過的或否認過的花色上的第一輪控制. 避免扣叫同伴自然叫出的花色上的單張或缺門, 以免誤導同伴以為你有支持.

    4. 叫回已經同意的王牌套(如可能, 在進局之前)來表示不鼓勵滿貫試探. 同伴高于進局叫品以上的扣叫表示強烈的滿貫興趣, 如你沒有滿貫興趣, 你必須叫回已同意的王牌套, 其它任何叫品都表示接受同伴的滿貫邀請, 可能的話大滿貫的誘惑. {見以下的 大滿貫迫叫(GSF)和第二部分的 5階高花自愿叫}

    5. 如果控制張扣叫被加倍, 再加倍則表示在那門花色上的第二輪控制. Pass則是鼓勵同伴 (叫回已同意的王牌則是阻止同伴).

    6. 在5階王牌花色以上的控制張扣叫則是大滿貫嘗試.

    7. 如果同伴中的任何一人能夠正確判斷出最後的定約, 他應該盡快地封叫.

    8. 當一方作出控制張扣叫, 同伴間就應該確認要最後停止在雙方所同意了的王牌花色上.

    阻截性叫品

    阻截性叫品包括弱2 (2♣除外)/3開叫或競叫表示有 6/7合理質量的套并有5-11點. 如果這手牌中有4張高花的話, 不應該采用阻截性叫品. 阻截性叫品的本意是阻截對方, 而不是同伴. 最好等同伴pass後再作阻截性開叫或競叫.

    注: 4♥ 或 4♠ 開叫也是阻截性開叫, 但要求有比弱2或弱3要更強一些的牌并有好的牌型. 加倍對方的4♥/4♠ 開叫是技朮性的加倍, 但常常可以被pass如果你認為懲罰更有利可圖的話. 5♣/5♦ 開叫也是阻截性的, 但加倍這樣的開叫一般是想要懲罰.

    對阻截性開叫的應叫

    在3階阻截叫後, 同伴應pass除非他有15+ 點.
    在2階阻截叫後:

    1. 2NT 應叫是迫叫并有進局的興趣(甚至對方加入了叫牌). 開叫方要交出有A或K的第二個套. 如沒有邊張的A或K, 就再叫自己的阻截叫套或加叫到3NT. {又見第二部分的奧咖斯特}
    2. 3NT 是止叫.
    3. 加叫同伴的花色是邀叫性的但不迫叫. {見 對弱二開叫加叫不迫叫}
    4. 新花叫品應叫(5+套) 迫叫一輪. 開叫方可以叫一個4張套的低花 (至少以Q打頭) 或叫 NT 來表示 8-11 點的牌, 支持同伴的花色 (如有 3 張或兩張帶大牌), 再叫自己的套 (表示低限 5-8 點).

    對弱二開叫加叫不迫叫 (RONF)

    這是一種用2NT和新花作為迫叫叫品的對弱二開叫的答叫方法. 因此, 加叫同伴的花色是唯一的不迫叫的應叫叫品.

    不尋常 NT( ! ) *

    　

    1. 在對方一階開叫後, 跳競叫 2NT 是 不尋常NT, 表示至少 5/5 的對方未叫過的雙低套(假設對方的叫品是自然叫品). 一般被用作弱牌防守性叫品 (8-10點雙套牌).
    2. 在對方開叫強2♣後, 2NT 也是不尋常NT, 但要有 16-19 點.
    3. 跳競叫2NT 後再加叫同伴所叫的花色, 或扣叫來表示在對方所叫的套上有控制, 都表示強牌 (16-18 點).

    對不尋常 2NT 的應叫:

    1. 示選叫品
    2. 跳叫性示選叫品 (一般是阻截性的)
    3. 扣叫 逼局或滿貫邀請
    4. 叫另一花色 (不迫叫)
    5. 叫3NT 如在另外兩個花色上有好的止張并在明手的長套上有足夠的進手
    6. 叫4NT是 黑木

    注 1: 除 3NT 外, 在對方兩人分別開叫應叫, 同伴又pass了的情況下, 其它任何 NT 叫品都是 不尋常NT.
    注 2: 在對方一階開叫後的 4NT 叫品也是不尋常NT.

    3. 如果技朮性加倍者在前面已經pass過, 技朮性加倍只保証 9-11 點并對未叫過的花色有4張支持.
    4. 如有 17+點, 技朮性加倍可用于單套或多套牌 (如在所有的花色上有止張, 可叫 NT). 技朮性加倍後再叫出自己的花色套表示強牌(加倍出套).
    5. 技朮性加倍後的跳叫是迫叫性叫品.
    6. 技朮性加倍後的4NT是 黑木問叫.

    扣叫對方花色 21+點, 滿貫興趣

    負加倍 (Negative X) ( ! )

    在同伴開叫, 右手方對手競叫到1階或2階後的加倍是負加倍, 表示有一定的牌力, 一般是6+HCP, 目的是給同伴留下更多的叫牌空間.
    注: 叫2階或更高階高花表示有 11+點 5 張以上套.

    負加倍的使用

    同伴 1 對方 1 同伴 2 同伴 2 表示牌情
    1♦ 1♥ X! 6+HCP 并有4張♠. 直接叫 1♠ 則保証有5張
    1♦ 1♠ X! 6+點并有4張 ♥ 或 5-10 點并有5+♥
    1♣ 1♦ X! 6+點并有4/4+ 高花
    1♥ 1♠ X! 4/4+ 低花

    注: 對對方1NT開叫的直接位置加倍一般是懲罰性加倍.

    對負加倍的應叫

    叫品 表示牌情
    最便宜的再叫 (未進局) 不超過16HCP (不迫叫)
    跳叫 16-18 點 (不迫叫)
    扣叫對方花色 19+點 (迫叫進局)
    Pass (極少出現) 懲罰性

    注 1: 除扣叫對方花色外, 負加倍者的任何未進局的再叫都是不迫叫的.
    注 2: 當采用負加倍後, 開叫者的同伴就不能在對方競叫後作懲罰性加倍了. 因此, 當對方競叫後緊接著兩個pass, 開叫者在對方的花色上只有2張或更少的牌張, 開叫者應試圖用加倍來重開叫, 因為同伴可能有一手很好的適合懲罰對方的牌, 但他在前面不能做懲罰性加倍.

    懲罰性加倍(Penalty X)

    如果你們的約定是負加倍上至 2♠, 以下的加倍就可能是懲罰性加倍:

    1. 加倍對方3階定約 (加倍對方的阻截性開叫除外, 那是技朮性加倍, 見上面的技朮性加倍).
    2. 直接加倍對方的 NT 開叫.
    3. 直接加倍對方的 NT 競叫.
    4. 在同伴已經準確的描述了他手上牌的強度和牌型後的加倍.
    5. 在本方任何一人在前面的叫牌中做過再加倍後的加倍.
    6. 在本方任何一人在前面的叫牌中做過懲罰性加倍或者pass過技朮性加倍後的加倍.
    7. 對對方的任何虛叫(例如: 斯台曼,扣叫, 黑木問叫的應叫). 這樣的加倍也是示攻性加倍.
    8. 對對方低花進局開叫的加倍.
    注 1: 在作懲罰性加倍前, 要慎重考慮同伴的競叫可能是基于不強的牌力.
    注 2: 除非另有約定, 對方的任何競叫或加倍後, 我方原本用于無競叫情況下的約定叫都要被取消了.
    注 3: 如果對方使用一個約定叫(例如 麥克爾扣叫 或 不尋常NT), 你可以加倍對方的虛叫來表示有至少 10HCP 或扣叫對方想要表示的花色來迫叫進局.
    注 4: 當對方明顯的出于阻截的目的在競叫, 你不知道是該加倍對方還是自己方叫的更高(一般已經超過了進局的級別). 可采用迫叫性 pass 來迫使同伴去加倍或再叫.
